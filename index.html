LENGUAJES Y HERRAMIENTAS

GitHub Es una plataforma para crear proyectos abiertos que se caracteriza por sus funciones colaborativas para mejorar el código. Como buen repositorio, el código de los proyectos que sean abiertos puede ser descargado y revisado por cualquier usuario, ayudando a mejorar el producto y crear ramificaciones a partir de él.

Git hoy en día es, con diferencia, el sistema de control de versiones moderno más utilizado del mundo. Git es un proyecto de código abierto maduro y con un mantenimiento activo que desarrolló originalmente Linus Torvalds, el famoso creador del kernel del sistema operativo Linux, en 2005.

Adobe Photoshop es un editor de fotografías, El hecho de que Photoshop sea el programa de retoque fotográfico más usado del mundo se debe a que este es una de las herramientas más completas de su categoría, con un número de posibilidades casi ilimitado. Una de las características más representativas; el sistema de capas.

InkScape es uno de los programas de diseño más populares entre los usuarios. Similar en posibilidades a reconocidos programas como Illustrator, Freehand o CorelDraw, la aplicación InkScape de código abierto, es un editor de gráficos vectoriales, que puede ser utilizado a nivel profesional.

Bootstrap es un framework de desarrollo web gratuito y de código abierto. Está diseñado para facilitar el proceso de desarrollo de los sitios web responsivos y orientados a los dispositivos móviles, proporcionando una colección de sintaxis para diseños de plantillas. En otras palabras, Bootstrap ayuda a los desarrolladores a construir sitios web más rápidamente, ya que no tienen que preocuparse por los comandos y funciones básicos.
